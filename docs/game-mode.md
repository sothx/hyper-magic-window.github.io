# 游戏显示布局

## 使用方式

模块 2.05.31+ 版本起，Web UI 支持配置游戏显示布局，可以改变游戏比例，获得更大的游戏视野。

Hyper OS 2.0+ 需要搭配修改版的手机管家/平板管家才会生效：

-  [移动云盘下载](https://caiyun.139.com/m/i?135CeBMHACC6p)

## 如何使用

通过模块 Web UI 即可配置任意游戏的显示布局，游戏应用必须添加到游戏工具箱才可以进行配置。
