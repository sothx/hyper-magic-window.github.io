# 常见问题（FAQ）

本文整理了《完美横屏应用计划》的常见问题。安装模块前，建议先阅读本页说明，很多安装、使用、适配和反馈问题都可以在这里找到答案。

## 一、使用问题

### 1. 模块目前适配了哪些应用？

模块目前适配的应用接近 8000 个，其中 700+ 高频应用经过手工精适配。安装《完美横屏应用计划》后，小米平板/折叠屏的平行窗口适配数量会显著增加。

模块还对少部分 Android 平板体验不佳的应用做了二次修正，例如：

1. 修正 QQ 在进入部分未适配平板的页面时会转为竖屏的问题，例如 QQ 群文件、图片编辑等页面。
2. 修正百度网盘在平板强制横屏后体验不佳的问题。

### 2. 模块对设备有什么要求？

《完美横屏应用计划》强依赖小米提供的反向适配 API，因此设备系统需要满足以下条件：

- Android 11 及以上；
- MIUI For Pad/Fold 或 Hyper OS For Pad/Fold；
- 已解锁 BL 并完成 ROOT，且能够安装 Magisk 模块。

暂不支持其他系统或其他品牌设备。

### 3. 未来会适配哪些设备？

目前主要适配 MIUI / Hyper OS，因为作者日常使用的是小米平板。

如果你希望模块支持其他设备，欢迎通过 GitHub 贡献相关代码。

### 4. 模块要怎么安装和使用？

《完美横屏应用计划》是 Magisk 模块。安装前，你需要具备一定动手能力，完成平板/折叠屏的 BL 解锁、ROOT 和 Magisk 模块安装。建议先熟悉标准 Magisk 模块安装流程，再尝试刷入本模块。

模块分为多个版本，请根据系统版本和设备型号选择合适的安装包。版本说明和下载地址见：

[https://hyper-magic-window.sothx.com/module-download.html](https://hyper-magic-window.sothx.com/module-download.html)

模块安装完成后，应用适配规则会自动扩充到系统设置中。不同系统入口如下：

- Android 11 平板：设置 - 横屏模式
- Android 12+ 平板：设置 - 平板专区 - 平行窗口，或 设置 - 平板专区 - 应用显示布局，或 设置 - 平板专区 - 应用横屏布局
- Android 12+ 平板：设置 - 平板专区 - 应用布局优化
- Android 12+ 折叠屏：设置 - 折叠屏专区 - 应用显示布局
- Android 12+ 折叠屏：设置 - 平板专区 - 应用布局优化

Android 12+ 模块安装后，可以在上述入口中自由开关应用的显示布局。

Android 11 模块安装后，所有适配应用的横屏模式会强制开启，无法在系统设置中逐个关闭。如需调整体验不佳的应用，可以使用模块 V13 版本提供的[自定义规则](https://hyper-magic-window.sothx.com/custom-config.html)。

### 5. 哪些应用是模块无法适配的？

以下类型的应用通常无法获得理想的平行窗口适配效果：

1. 基于 Flutter、React Native 或网页套壳的 App。由于多数页面共用同一个 Activity，除非应用主动适配平板，否则被动适配大多体验不佳。这类应用通常只能使用信箱模式或强制横屏，例如酷我音乐、酷狗音乐概念版等。
2. 已经主动适配平板的 App。对于微信、酷安、微博、豆瓣等应用，模块通常无法再介入平行窗口相关规则。Hyper OS 2.0 起，可以通过 `skipSelfAdaptive="true"` 强制部分应用走系统平行窗口适配规则。

### 6. 安装模块后为什么仍然不支持左右滑动调节？

左右滑动调节与系统版本、设备型号和模块版本有关，请先确认以下情况：

1. 小米仅在 Android 14+ 的 Hyper OS For Pad 中全量下放左右滑动调节。请确保系统至少为 Android 14，再安装对应模块。
2. 小米平板 5 / 小米平板 5 Pro / 小米平板 5 Pro 5G 如果已升级到官方基于 Android 13 的 Hyper OS（非官改、非移植包），可以尝试安装“小米平板5系列安卓13澎湃专版”。该版本可支持左右滑动调节，安装前请自备救砖模块。
3. Android 13 下，仅 Xiaomi Pad 6 Max 原生支持左右滑动调节。如果你的系统是基于 Xiaomi Pad 6 Max 的 MIUI 14 第三方移植 ROM，可以尝试安装“小米平板6系列MIUI14专版”。该版本可支持左右滑动调节，安装前请自备救砖模块。
4. 并不是所有应用都会启用左右滑动调节。部分应用启用后体验较差，因此模块会主动移除这些应用的左右滑动调节适配。

### 7. 可以让只有横屏的应用在竖屏时使用吗？例如哔哩哔哩 HD。

可以。通过模块 V13 版本提供的[自定义规则](https://hyper-magic-window.sothx.com/custom-config.html)，可以让哔哩哔哩 HD 这类只有横屏布局的应用在竖屏时使用。

### 8. 我想使用较老版本的应用，但当前规则不适配老版本，该怎么办？

可以通过模块 V13 版本提供的[自定义规则](https://hyper-magic-window.sothx.com/custom-config.html)，自行覆盖或调整对应应用的适配规则，使其兼容老版本应用。

### 9. 可以给模块里没有的应用添加强制横屏吗？

可以。通过模块 V13 版本提供的[自定义规则](https://hyper-magic-window.sothx.com/custom-config.html)，可以很方便地为应用添加强制横屏规则。由于这是系统级强制横屏，稳定性通常会优于 Rotation 这类第三方旋转工具。

### 10. 哪些应用有更好的替代版本？

大部分影音应用都有 HD 版、TV 版、音箱版或车机版。建议优先使用这些面向大屏设备的版本，它们通常拥有更好的横屏和大屏体验。

### 11. 为什么任务栏和小白条无法沉浸？

这是系统官方不支持的行为，模块无法处理。

### 12. 为什么从任务栏点进去的消息不能全屏？

这是系统官方不支持的行为，模块无法处理。

### 13. 为什么我的滑动条不能左右调整比例？我用的是 Mix Fold 系列。

这是系统官方不支持的行为，模块无法处理。

### 14. 为什么有时候过渡动画很奇怪，窗口会旋转一圈，或者黑屏一下？

这是小米系统本身的 BUG，需要等待小米官方修复，模块无法从根本上解决。

### 15. 如何提交适配缺陷或适配需求？

可以通过 [GitHub Issues](https://github.com/sothx/mipad-magic-window/issues)提交应用适配缺陷或适配需求。

提交前请先确认：

1. 模块已经升级到最新版；
2. 问题在最新版中仍然存在；
3. 描述中包含设备型号、系统版本、模块版本、应用名称和复现步骤。

### 16. 为什么模块有时会失效？

正常情况下，模块已内置较完善的防云控策略，不应该被系统云控覆盖。如果仍然发现规则失效，请通过 GitHub Issues 主动反馈，方便后续排查和修复。

### 17. 可以像 MatePad 那样固定左侧/右侧页面，或者在小窗模式下使用平行视界吗？

不可以。小米遵循的是 Android 12L 的 Embedded Activity 规范，仅支持点击左侧页面后在右侧展开二级页面，不支持固定某一侧页面，也不支持在小窗模式下展开平行视界。

华为使用的是自研 Magic Window 规范，与 Android 12L 的 Embedded Activity 规范各有优劣。由于华为方案可定制性更强，因此可以做到左侧或右侧单独固定页面，也可以在小窗模式下展开平行视界。

### 18. 如何卸载模块？

模块 V13 起已简化卸载流程：在 Magisk 中移除本模块，重启后即可恢复为设备出厂规则。

模块会锁定部分系统文件，防止系统云控覆盖模块规则。由于这一机制，系统升级时可能因权限不足导致卡米。建议在系统升级前先卸载模块，升级完成后再按需重新安装。

## 二、应用适配

### 1. 安装模块后为什么京东变全屏了？

新版京东已经开始自行适配平板。请确保京东已升级到最新版。

首次打开京东时，应用可能尚未触发平板判断；关闭后第二次打开，通常即可正常使用。

### 2. 为什么抖音不能全屏了？

如果使用了改机型模块，可能导致抖音评论区丢失横屏。可以通过模块 V13 版本提供的[自定义规则](https://hyper-magic-window.sothx.com/custom-config.html)，为抖音添加强制横屏规则。

- Hyper OS 2.0

```xml
<!-- /data/adb/MIUI_MagicWindow+/config/fixed_orientation_list.xml-->
  <!--抖音增加强制横屏的选项-->
  <package name="com.ss.android.ugc.aweme" supportModes="full,fo" defaultSettings="full" />
  <!--抖音火山版增加强制横屏的选项-->
  <package name="com.ss.android.ugc.live" supportModes="full,fo" defaultSettings="full" />
  <!--抖音极速版增加强制横屏的选项-->
  <package name="com.ss.android.ugc.aweme.lite" supportModes="full,fo" defaultSettings="full" />
```

- MIUI 13 到 Hyper OS 1.0

```xml
<!-- /data/adb/MIUI_MagicWindow+/config/embedded_rules_list.xml -->
  <!--抖音强制横屏-->
  <package name="com.ss.android.ugc.aweme" fullRule="*" />
  <!--抖音火山版强制横屏-->
  <package name="com.ss.android.ugc.live" fullRule="*" />
  <!--抖音极速版强制横屏-->
  <package name="com.ss.android.ugc.aweme.lite" fullRule="*" />
```

### 3. 安装模块后，为什么抖音评论区不能显示在侧边？

该问题与模块无关。抖音评论区有自己的云控规则。请先确保未安装改机型相关模块，再尝试以下操作：

1. 清除抖音数据。
2. 进入 平板管家 - 虚拟身份管理 - 虚拟身份授权 ID 管理，关闭抖音。
3. 重新打开抖音，结束抖音后台后再次打开。第二次打开时，评论区通常会恢复正常。

### 4. 微信的左右滑动调节会导致动画闪烁，怎么去掉？

可以通过模块 V13 版本提供的[自定义规则](https://hyper-magic-window.sothx.com/custom-config.html)，移除微信的左右滑动调节。

### 5. 为什么微信没有分屏了，点击聊天界面还会变成竖屏？

该问题通常与本模块无关，大概率是微信没有识别到当前设备为平板。请检查是否安装过修改 PPI、伪装机型或修改系统功能类的模块。

目前发现以下模块可能导致微信识别设备类型异常，请谨慎使用：

- Moto [DolbyAtmos] A14 v1.0a

### 6. 智能助理添加的快递小组件，点进淘宝后只有右侧界面且会卡死，怎么解决？

新版淘宝已经自行适配平板，模块不再对淘宝适配施加干预。该问题来自淘宝自身的平板适配缺陷，建议向淘宝官方反馈并等待修复。

### 7. 为什么哔哩哔哩看直播无法全屏？还有一些应用界面显示不全。

如果你使用的是 Android 13，请将模块更换为 Android 13 专版。该版本针对部分不支持左右滑动条的机型做了兼容适配。

如果其他应用也出现类似问题，且更换专版后仍未解决，可以提交反馈。

### 8. 为什么 X 一打开就会崩溃？

该问题原因暂未明确，目前主要发现于 Xiaomi Pad 6S Pro。如果遇到 X 打开即崩溃，建议使用 Xiaomi Pad 6S Pro 专版模块。该专版会将 X 的适配方式改为强制横屏。

### 9. 为什么医考帮跟其他应用分屏后会重载？

医考帮自身已经适配平板。模块只是修正了医考帮从分屏恢复全屏后会丢失平板全屏化的问题。分屏后页面重载属于医考帮自身问题，模块无法解决，建议向医考帮官方反馈。

### 10. 为什么 QQ/微信视频通话时，对方看到的画面会旋转 90 度？

该问题与模块无关，属于 QQ/微信长期存在的 App 适配设计问题，并非系统问题。请向对应 App 开发者反馈。

### 11. 为什么闲鱼不是默认分屏？官方适配是默认分屏。

闲鱼默认展开分屏后会出现页面崩溃 BUG。模块适配优先考虑实用性和稳定性，因此取消了闲鱼的默认分屏。

### 12. CCTalk 为什么没有适配？小米应用商店页面显示它有适配。

CCTalk 自身平板适配并不完善，进入视频播放页后可能无法正常退出。同时，由于 CCTalk 的部分历史代码问题，它也无法被系统正常反向适配。

如果需要 CCTalk 的平行窗口，需要将模块升级到至少 1.23.88，并安装提供的修改版 CCTalk 客户端。

修改版 CCTalk 安装包无签名，需要搭配核心破解使用，也可以使用 HyperCeiler 内置的核心破解。

> 修改版客户端可能存在被 CCTalk 封号的风险，请自行判断是否使用。

修改版 CCTalk 下载链接：https://caiyun.139.com/m/i?135Cm8YnuNKOg
