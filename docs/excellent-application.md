# 精选应用

《完美横屏应用计划》不仅扩充了大量平行窗口的适配数量，还针对部分体验不佳的应用进行了精适配，这里罗列了一些经模块适配后表现良好的应用作为精选推荐应用。

更新日期: 2024/09/08

## 社交类

### QQ(com.tencent.mobileqq)

适配兼容性: 

Android 13+ | MIUI For Pad | Hyper OS For Pad

适配简述:

1.纠正了QQ部分场景界面会扭转为强制竖屏的问题(如群文件、图片编辑)

2.纠正了QQ视频通话摄像头转向错误的问题
